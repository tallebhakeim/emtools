🔌 Circuit — SPICE light

Un mini-simulateur de circuit dans le navigateur : décris un réseau R / L / C et sources, lance l'analyse fréquentielle (Bode) ou transitoire. Résolu par analyse nodale modifiée (MNA), comme SPICE.
Comment ça marche — écris une netlist (un composant par ligne, syntaxe façon SPICE), choisis un nœud de masse 0, puis lance l'analyse. En fréquentiel, la source AC sert d'entrée et on trace le diagramme de Bode (gain en dB + phase) de chaque nœud. En transitoire, on intègre la réponse temporelle (sources DC en échelon ou SIN). Le moteur résout le circuit par analyse nodale modifiée à chaque fréquence / pas de temps.
Circuit
Clique un outil puis le quadrillage pour poser un composant (orientation via ⟳). Outil « Fil » : clic-clic entre deux points. Les bornes qui se touchent (ou reliées par un fil) forment un même nœud. Place une Masse = nœud 0. Double-clic / sélection pour éditer une valeur.
Aide-mémoire syntaxe
R1 a b 1k · C1 a b 100n · L1 a b 10m — résistance / condensateur / bobine entre nœuds a et b.
V1 in 0 AC 1 — source de tension (entrée fréquentielle, amplitude 1, phase optionnelle AC 1 90).
V1 in 0 DC 1 — échelon ; V1 in 0 SIN(0 1 1k) — sinus (offset, amplitude, fréquence) pour le transitoire.
I1 a b AC 1 — source de courant. K1 L1 L2 0.5 — couplage mutuel entre deux bobines (transfo / WPT ; M = k·√(L1·L2)). D1 a k — diode (anode a → cathode k ; modèle Shockley, options Is n). Masse = nœud 0. Suffixes : f p n u m k meg g.
Analyse
Réponse — diagramme de Bode
Outil pédagogique — circuits linéaires R, L, C et sources indépendantes (V, I), résolus par analyse nodale modifiée : exact en régime établi (AC) et par Euler implicite en transitoire ; couplage mutuel K (transformateur / WPT) et diode (Shockley, Newton-Raphson) supportés. Pas encore de transistor ni de sources commandées. Donne les ordres de grandeur et la physique, pas un signoff industriel.
