🛰 Réseau planaire 2D

Réseau M×N : dépointage du faisceau en azimut + élévation, carte de gain (plan UV), lobes de réseau, directivité.
Principe — une grille planaire de M×N éléments (espacements dx, dy) avec des déphasages progressifs βx, βy pointe le faisceau dans une direction (θ₀, φ₀) telle que βx = −k·dx·sinθ₀·cosφ₀ et βy = −k·dy·sinθ₀·sinφ₀. Le facteur de réseau se sépare : AF = AFx·AFy. On visualise le gain dans le plan UV (u = sinθ·cosφ, v = sinθ·sinφ) : le faisceau est une tache qui se déplace avec le dépointage.
